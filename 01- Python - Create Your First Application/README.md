# 01 - Python Web: First Application

This module is part of the **Alura Python Backend Developer** track and introduces the fundamentals of building a Python application, focusing on programming logic and basic system structure.

## Module objectives

- Create a Python project using VS Code
- Understand application flow using conditionals and loops
- Work with error handling using `try-except`
- Create functions to structure the program
- Build an interactive main menu
- Manage data using lists and dictionaries

## Project developed

During this module, a simple **restaurant registration and management system** was built, applying the concepts learned in practice.

## Topics covered

- Basic Python project structure
- Conditionals (`if`, `elif`, `else`)
- Loops (`for`, `while`)
- Exception handling
- Functions
- Data structures (lists and dictionaries)

## Goal

To strengthen Python fundamentals by building a simple, interactive, and structured application.
